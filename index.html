// правила
Как это работает

Простые правила — чтобы каждый получил максимум.

01

Приходи с идеей

Бот, сайт, автоматизация, SaaS — что угодно. Нет идеи — найдём на месте.

02

Работай весь день

Это не лекция. Ты реально кодишь с 11 до 19. В конце — демо рабочего прототипа.

03

Помогай другим

Застрял — попроси помощь. Видишь что кто-то застрял — подойди.

04

Покажи результат

3 минуты на демо. Экран, без слайдов. Работающий продукт > презентация.

05

Используй AI

Claude, ChatGPT, Cursor, Bolt — любые инструменты. Строим, а не обсуждаем.

06

Запиши демо

Записывай экран во время демо. Видео загружается на платформу — строим портфолио сообщества.

Чеклист
  • Проверь доступ к lovable.dev заранее
  • Идеи — в таблицу
  • Демо с видео — в Топик
Не ок
  • Тусить / отвлекать, не вайбкодя
    Исключение: эксперт для custdev или помощь с функциями
Рекомендуем
  • Говорить промпт в микрофон
  • Начинать с lovable.dev → даже опытным
    → потом VS Code + Opus
